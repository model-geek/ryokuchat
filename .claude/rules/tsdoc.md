# TSDoc ルール

> 詳細: docs/tsdoc.md

## 基本

- 日本語で統一。全要素に記載（公開・内部問わず）
- インラインコメント（//）禁止 → 内部変数に TSDoc を付与

## 形式

- データ型フィールド: 単行 /** コメント */
- それ以外: 複数行コメント
- フィールド間に空行必須
- 説明文と @タグの間に空行。@タグ同士は空行なし

## 内容

- null / undefined のドメイン意味は @remarks で明記
- リテラルユニオンの各値は @remarks で説明
- TypeScript の値はバッククォートで囲む

## 文体

- です・ます調
- 英単語・半角数字と日本語の間に半角スペース
- 補足は半角括弧 (補足)。全角括弧は不可
- ドメイン用語はユビキタス言語に統一（表記揺れ禁止）
