# RyokuChat

オープンソースのセルフホスティングチャットアプリ。

## Features

- リアルタイムメッセージング
- チャンネルベースの会話
- PWA 対応

## Tech Stack

- Next.js App Router
- Park UI (Panda CSS)
- Drizzle ORM + Supabase
- Serwist (PWA)
