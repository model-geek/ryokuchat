# RyokuChat

オープンソースのセルフホスティングチャットアプリ。

## Features

- リアルタイムメッセージング
- チャンネルベースの会話
- PWA 対応

## Tech Stack

| Layer | Technology | Version |
|---|---|---|
| Framework | Next.js App Router | 16.1.6 |
| UI | Park UI (Panda CSS) + Ark UI | 1.0.1 / 1.6.0 / 5.31.0 |
| ORM | Drizzle ORM + postgres.js | 0.45.1 / 3.4.8 |
| Backend | Supabase (PostgreSQL, Auth, Realtime) | 2.97.0 |
| PWA | Serwist (@serwist/next) | 9.5.6 / 9.5.4 |
| E2E Test | Playwright | 1.58.2 |
